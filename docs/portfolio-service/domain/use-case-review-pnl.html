Use Case

UC-REVIEW-PNL — Review Total PnL

UC-REVIEW-PNL · Updated 2026-06-02 · Source root services/portfolio-service

Investor retrieves their total portfolio profit and loss.

UC-REVIEW-PNLuse-casedocumented

Key Facts

ActorInvestor (via API Gateway)
GoalRead total PnL
TriggerUser opens PnL view (GET /portfolio/pnl?userId=...)
PreconditionsAuthenticated user
PostconditionstotalPnl returned (calculation method unspecified)
Business FlowBFLOW-PNL-REVIEW
TraceabilityTRACE-PORTFOLIO

Main Scenario

  1. User requests GET /portfolio/pnl with userId query parameter.
  2. Controller calls getTotalPnl(userId).
  3. Service returns HTTP 200 with an object containing totalPnl.

Alternate and Exception Paths

Current implementation returns a fixed value; the calculation method and market-price integration are unspecified.
