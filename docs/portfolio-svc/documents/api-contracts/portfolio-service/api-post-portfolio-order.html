API-002: POST /portfolio/order

Service: portfolio-service Status: Partial

HTTP Contract

MethodPOST
Path/portfolio/order
Request BodyJSON order object.
Success Status201 Created

Request Body

FieldRequiredTypeDescription
userIdYesUUID stringOwner of the order.
symbolYesStringStock symbol.
typeYesStringBUY or SELL.
quantityYesIntegerOrder quantity.
priceNoNumberLimit price.

Example Request

{
  "userId": "4d7f77a0-94a6-4e9d-9f3e-7dd27d15a9ef",
  "symbol": "AAPL",
  "type": "BUY",
  "quantity": 10,
  "price": 175.50
}

Response Shape

{
  "order_id": "UUID",
  "status": "PENDING",
  "message": "Lệnh đặt thành công, đang chờ khớp"
}

Side Effects

  • Validates price band, available cash/holdings, and trading hours before accepting the order.
  • Freezes the corresponding cash or shares.
  • Persists the order/transaction and publishes an event (MSG-002).
