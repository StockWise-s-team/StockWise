MSG-003: Order Created Event

RabbitMQ Status: Implemented

An outbound event published by the portfolio service immediately after a new pending order is created and saved, per FR-PORTFOLIO-04.

Contract Summary

Exchangeorder.exchange
Exchange Typetopic
Routing Keyorder.created
Producerportfolio-service (this service)
DirectionPublished to notify downstream systems (e.g. matching engine) of a new order.
