Repository

  • TransactionRepository extends JpaRepository<Transaction, UUID>.
  • History lookup: findByPortfolioIdOrderByExecutedAtDesc(UUID portfolioId).

Append-Only Constraint

  • Per FR-PORTFOLIO-07 / NFR-REL-03, transactions are append-only — never updated or deleted.
  • This entity holds executed BUY/SELL records. The SRS also describes an order with a status lifecycle (PENDING/CANCELLED), modelled separately as ENTITY-003.
