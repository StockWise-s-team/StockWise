Runtime Notes

  • Hibernate is configured with ddl-auto=update; local Compose also mounts infra/postgres/init.sql into the Postgres initialization directory.
  • The service binds portfolio_service_price_q to market.exchange with routing key price.updated. See MSG-001 for the messaging contract.
  • Actuator exposes health and info through the management web endpoint configuration.
