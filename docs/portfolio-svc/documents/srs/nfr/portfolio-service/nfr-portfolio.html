NFR: Portfolio Service Non-Functional Requirements

System-wide non-functional requirements from the StockWise SRS that constrain the portfolio-service, each with a simple implementation status.

NFR-SCA-01 The microservices architecture allows each service to scale independently. Implemented

NFR-USA-03 Every error state has a clear message for the user. Implemented
